CORE STYLE RULES

The hook should feel conversational and slightly sarcastic.

   2. NARRATION STYLE
Write in casual spoken English.
Use phrases like:
      * “Now here’s the thing…”
      * “But unfortunately…”
      * “And that’s where things go wrong.”
      * “Until.”
      * “And then things get worse.”
      * “Bro was basically…”
      * “Like seriously…”

      * “Because apparently that wasn’t enough for them.”
The narration MUST feel natural and human.

      3. STORY STRUCTURE
Follow this exact pacing structure:
PHASE 1 — INTRODUCTION
         * Introduce main criminal
         * age
         * location
         * personality
         * lifestyle
         * motivations
         * greed / obsession / ego / jealousy etc.
PHASE 2 — SETUP
         * Explain how they entered the situation
         * relationships
         * businesses
         * marriage
         * affair
         * opportunity
         * early warning signs
PHASE 3 — ESCALATION
         * show increasingly bad decisions
         * manipulation
         * scams
         * lies
         * coverups
         * abuse
         * greed
         * crimes getting worse over time
PHASE 4 — “UNTIL”
At some point write:
“Until.”
OR
“Until one day…”
This is the major turning point.

         4. BUILD TENSION SLOWLY
The audience should slowly realize:
“Oh this person is actually evil.”
Do NOT reveal everything immediately.
